## Practicies
### Branches:
<b>master</b>  - branch pre finálne verzie bez bugov  
<b>develop</b> - branch pre development, najviac sa bude pracovať tu  
<b>feature_*</b>  - branche pre nové features, ktoré keď budú dokončené budú pushnuté do develop
